# 未来软件工作室数字化平台-API
技术栈：SpringBoot、SpringMVC、MyBatis、Redis、MySQL、RabbitMQ、Minio、SpringSecurity

## 简介
  本平台将博客、考勤、课表、成员信息等分散数据统一整合、集中管理，解决了团队长期依赖 CSDN、钉钉考勤、线下
统计等多平台分散、效率低、数据不互通的问题，完善了团队的信息化管理体系，大幅提升内部协作与运营效率。 

## 项目开始

### 项目打包

```bash
$ mvn package
```

### 项目运行
```bash
$ docker-compose -d
```
